# 3D Gaussian Splatting 模型文件与可视化指南

## 训练产生的模型文件

基于刚才的训练测试，模型输出目录为：`output/gaussian_model_20260414_134456/`

### 主要模型文件

```
output/gaussian_model_20260414_134456/
├── cameras.json              # 相机参数文件
├── cfg_args                  # 训练配置参数
├── exposure.json             # 曝光参数（如果使用曝光补偿）
├── input.ply                 # 初始点云（来自COLMAP）
├── point_cloud/              # 训练过程中的点云文件
│   └── iteration_100/        # 第100次迭代的结果
│       └── point_cloud.ply   # 主要的3D高斯模型文件（45MB）
├── training_config.txt       # 训练配置
└── training_report.md        # 训练报告
```

### 关键文件说明

#### 1. **point_cloud.ply** (主要3D高斯模型)
- **格式**: PLY (Polygon File Format)
- **大小**: 45MB (100次迭代后)
- **内容**: 包含所有3D高斯的参数：
  - 位置 (x, y, z)
  - 颜色 (RGB)
  - 不透明度 (alpha)
  - 缩放 (scale_x, scale_y, scale_z)
  - 旋转 (四元数: qx, qy, qz, qw)
  - 球谐函数系数 (用于视角相关的外观)

#### 2. **cameras.json**
- 包含所有训练和测试相机的参数
- 相机位置、方向、焦距、图像尺寸等
- 用于渲染和可视化时确定视角

#### 3. **input.ply**
- 初始点云，来自COLMAP的稀疏重建
- 训练开始时作为初始化的基础

#### 4. **cfg_args**
- 训练时使用的命令行参数
- 可以用于复现训练过程

## 可视化方法

### 方法1：使用SIBR实时查看器（推荐）

#### 步骤1：构建SIBR查看器
```bash
cd SIBR_viewers
cmake -Bbuild .
cmake --build build --target install --config RelWithDebInfo
```

#### 步骤2：运行查看器
```bash
# 查看刚才训练的模型
./SIBR_viewers/install/bin/SIBR_gaussianViewer_app \
  -m output/gaussian_model_20260414_134456

# 或者使用完整路径
./SIBR_viewers/install/bin/SIBR_gaussianViewer_app \
  -m /home/not0382/Learning/3DGS/gaussian-splatting/output/gaussian_model_20260414_134456
```

#### 步骤3：查看器控制
- **鼠标左键拖动**: 旋转视角
- **鼠标右键拖动**: 平移场景
- **鼠标滚轮**: 缩放
- **W/A/S/D**: 移动相机
- **空格键**: 重置视角
- **F键**: 切换全屏
- **H键**: 显示帮助

## 模型文件格式详解

### PLY文件结构
```
ply
format binary_little_endian 1.0
comment Created by 3D Gaussian Splatting
element vertex 182686           # 高斯数量
property float x                # X坐标
property float y                # Y坐标
property float z                # Z坐标
property float nx               # 法线X（通常为0）
property float ny               # 法线Y（通常为0）
property float nz               # 法线Z（通常为0）
property float f_dc_0           # 球谐函数DC分量 R
property float f_dc_1           # 球谐函数DC分量 G
property float f_dc_2           # 球谐函数DC分量 B
property float f_rest_0         # 球谐函数剩余系数
...                             # 更多球谐系数
property float opacity          # 不透明度
property float scale_0          # 缩放 X
property float scale_1          # 缩放 Y
property float scale_2          # 缩放 Z
property float rot_0            # 旋转四元数 qw
property float rot_1            # 旋转四元数 qx
property float rot_2            # 旋转四元数 qy
property float rot_3            # 旋转四元数 qz
end_header
<二进制数据>
```

### PyTorch检查点文件 (.pth)
- 包含完整的模型状态
- 可以用于继续训练
- 包含优化器状态、学习率调度器等

## 可视化技巧

### 1. 调整渲染质量
在SIBR查看器中：
- 按 **1-9** 键：调整渲染质量（1=最低，9=最高）
- 按 **T** 键：切换透明度渲染
- 按 **L** 键：切换光照效果

### 2. 查看不同迭代的模型

```bash
  -m output/gaussian_model_20260414_134456/point_cloud/iteration_7000
```

### 3. 批量渲染图像
```bash
# 渲染所有训练视图
python render.py -m output/gaussian_model_20260414_134456 --skip_test

# 渲染所有测试视图
python render.py -m output/gaussian_model_20260414_134456 --skip_train

# 渲染特定视角
python render.py -m output/gaussian_model_20260414_134456 --specific_view 0 5 10
```

### 4. 创建视频漫游

## 故障排除

### 问题1：SIBR查看器无法打开
```bash
# 检查是否构建成功
ls SIBR_viewers/install/bin/

# 检查依赖
ldd SIBR_viewers/install/bin/SIBR_gaussianViewer_app

# 重新构建
cd SIBR_viewers
rm -rf build
cmake -Bbuild .
cmake --build build --target install --config RelWithDebInfo
```

### 问题2：PLY文件太大无法打开
```bash
# 使用简化版本
python -c "
import open3d as o3d
pcd = o3d.io.read_point_cloud('point_cloud.ply')
pcd_simplified = pcd.voxel_down_sample(voxel_size=0.01)
o3d.io.write_point_cloud('point_cloud_simplified.ply', pcd_simplified)
"

# 然后查看简化版本
./SIBR_viewers/install/bin/SIBR_gaussianViewer_app -m point_cloud_simplified.ply
```

### 问题3：渲染图像质量差
```bash
# 增加渲染分辨率
python render.py -m output/gaussian_model_20260414_134456 --resolution 2

# 使用抗锯齿
python render.py -m output/gaussian_model_20260414_134456 --antialiasing
```

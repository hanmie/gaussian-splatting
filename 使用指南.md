# 3D Gaussian Splatting 使用指南

## 环境配置

### 1. 激活 Conda 环境
```bash
conda activate gaussian_splatting
```

### 2. 设置编译器环境变量（解决 CUDA 兼容性问题）
```bash
# 临时设置（当前终端有效）
export CC=/usr/bin/gcc
export CXX=/usr/bin/g++
export CUDAHOSTCXX=/usr/bin/g++

# 永久设置（添加到 ~/.bashrc）
echo 'export CC=/usr/bin/gcc' >> ~/.bashrc
echo 'export CXX=/usr/bin/g++' >> ~/.bashrc
echo 'export CUDAHOSTCXX=/usr/bin/g++' >> ~/.bashrc
source ~/.bashrc
```

### 3. 安装 CUDA 扩展模块（如果未安装）
```bash
# 安装 diff-gaussian-rasterization
cd submodules/diff-gaussian-rasterization
pip install . --no-build-isolation
cd ../..

# 安装 simple-knn
cd submodules/simple-knn
pip install . --no-build-isolation
cd ../..

# 安装 fused-ssim
cd submodules/fused-ssim
pip install . --no-build-isolation
cd ../..
```

### 4. 验证安装
```bash
python -c "import simple_knn; import diff_gaussian_rasterization; import fused_ssim; print('所有CUDA扩展模块导入成功！')"
python -c "import sys; sys.path.insert(0, '.'); from gaussian_renderer import render; print('gaussian_renderer 导入成功！')"
```

## 数据集准备

### 1. 数据集结构要求
```
<数据集路径>
├── images/          # 图像文件夹
│   ├── image1.jpg
│   ├── image2.jpg
│   └── ...
└── sparse/          # COLMAP 稀疏重建结果
    └── 0/
        ├── cameras.bin
        ├── images.bin
        └── points3D.bin
```

### 2. 使用示例数据集
项目自带 Tanks & Temples 和 Deep Blending 数据集：
- `assets/tandt_db/tandt/train/` - Tanks & Temples 训练场景
- `assets/tandt_db/tandt/truck/` - Tanks & Temples 卡车场景
- `assets/tandt_db/db/drjohnson/` - Deep Blending 场景
- `assets/tandt_db/db/playroom/` - Deep Blending 游乐室场景

### 3. 处理自己的图像
```bash
# 1. 将图像放在 <路径>/input/ 目录下
# 2. 运行转换脚本
python convert.py -s <路径> [--resize]

# 示例
python convert.py -s my_dataset --resize
```

## 训练模型

### 1. 基础训练命令
```bash
python train.py -s <数据集路径>
```

### 2. 使用示例数据训练
```bash
# 简短测试（100次迭代）
python train.py -s assets/tandt_db/tandt/train --iterations 100 --model_path test_output

# 完整训练（默认30,000次迭代）
python train.py -s assets/tandt_db/tandt/train

# 指定输出目录
python train.py -s assets/tandt_db/tandt/train -m my_model_output
```

### 3. 带评估的训练
```bash
# 训练时保留测试集
python train.py -s <数据集路径> --eval

# 渲染测试结果
python render.py -m <模型路径>

# 计算评估指标
python metrics.py -m <模型路径>
```

### 4. 常用训练参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-s, --source_path` | 数据集路径（必需） | - |
| `-m, --model_path` | 模型输出目录 | `output/随机名称` |
| `--eval` | 使用训练/测试分割 | `False` |
| `--iterations` | 训练迭代次数 | `30000` |
| `-r, --resolution` | 图像分辨率（1=原始，2=一半，4=四分之一） | 自动调整 |
| `--white_background` | 使用白色背景 | `False` |
| `--data_device` | 数据存储设备（`cuda` 或 `cpu`） | `cuda` |
| `--sh_degree` | 球谐函数阶数 | `3` |
| `--position_lr_init` | 初始位置学习率 | `0.00016` |
| `--position_lr_final` | 最终位置学习率 | `0.0000016` |
| `--feature_lr` | 特征学习率 | `0.0025` |
| `--opacity_lr` | 不透明度学习率 | `0.05` |
| `--scaling_lr` | 缩放学习率 | `0.005` |
| `--rotation_lr` | 旋转学习率 | `0.001` |

### 5. 高级训练选项
```bash
# 使用加速优化器（需要安装加速版本）
python train.py -s <数据集路径> --optimizer_type sparse_adam

# 使用深度正则化
python train.py -s <数据集路径> -d <深度图路径>

# 使用曝光补偿
python train.py -s <数据集路径> --exposure_lr_init 0.001 --exposure_lr_final 0.0001 --exposure_lr_delay_steps 5000 --exposure_lr_delay_mult 0.001 --train_test_exp

# 使用抗锯齿
python train.py -s <数据集路径> --antialiasing
```

## 故障排除

### 1. 常见问题

#### Q: 编译 CUDA 扩展失败
**A**: 确保使用系统 GCC 而非 Conda GCC：
```bash
export CC=/usr/bin/gcc
export CXX=/usr/bin/g++
export CUDAHOSTCXX=/usr/bin/g++
```

#### Q: VRAM 不足
**A**: 尝试以下方法：
1. 降低分辨率：`-r 2` 或 `-r 4`
2. 减少迭代次数：`--iterations 7000`
3. 增加梯度阈值：`--densify_grad_threshold 0.001`
4. 使用 CPU 存储数据：`--data_device cpu`

#### Q: 训练速度慢
**A**: 启用加速优化器：
```bash
# 切换到加速分支
cd submodules/diff-gaussian-rasterization
git checkout 3dgs_accel
pip install . --no-build-isolation
cd ../..

# 使用稀疏 Adam 优化器
python train.py -s <数据集路径> --optimizer_type sparse_adam
```

### 2. 性能优化建议
1. **大场景**：降低学习率 `--position_lr_init 0.000016 --scaling_lr 0.001`
2. **高分辨率图像**：使用 `--data_device cpu` 减少 VRAM 使用
3. **快速测试**：设置 `--test_iterations -1` 避免测试时的内存峰值

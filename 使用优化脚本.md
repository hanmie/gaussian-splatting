# 3D Gaussian Splatting 模型优化脚本使用指南

## 可用脚本

### 1. `optimize_model_fixed.py` (推荐)
修复版优化脚本，正确处理Sigmoid/exp激活前的原始值。

### 2. `optimize_model.py` 
原始优化脚本（有已知问题，不推荐使用）。

## 参数详解

### 必需参数
- `input`: 输入PLY文件路径
  ```bash
  python optimize_model_fixed.py "path/to/your/model.ply"
  ```

### 可选参数

#### `--method`: 优化强度
- `aggressive`: 激进优化 - 显著减少阴影和不透明区域
- `moderate`: 中等优化 - 平衡优化（默认）
- `conservative`: 保守优化 - 轻微调整

#### `--max-scale`: 最大缩放限制
- 类型: float
- 默认: 0.5
- 说明: 限制高斯点的最大缩放（exp后的值）

#### `--remove-outliers`: 移除异常点
- 类型: flag
- 说明: 移除极低不透明度和极小缩放的点

#### `--opacity-threshold`: 不透明度阈值
- 类型: float
- 默认: 0.001
- 说明: 低于此阈值（Sigmoid后）的点将被移除（需配合--remove-outliers）

#### `--scale-threshold`: 缩放阈值
- 类型: float
- 默认: 0.001
- 说明: 低于此阈值（exp后）的点将被移除（需配合--remove-outliers）

#### `--output-suffix`: 输出文件后缀
- 类型: string
- 默认: "_optimized_fixed"
- 说明: 优化后文件的后缀

## 故障排除

### 错误1：缺少输入参数
```
error: the following arguments are required: input
```
**解决方案**：提供PLY文件路径
```bash
# 错误用法
python optimize_model_fixed.py

# 正确用法
python optimize_model_fixed.py "path/to/model.ply"
```

### 错误2：文件不存在
```
FileNotFoundError: [Errno 2] No such file or directory: '...'
```
**解决方案**：检查文件路径
```bash
# 确认文件存在
ls -la "output/tandt_train_20260414_135259/point_cloud/iteration_30000/"

# 使用正确路径
python optimize_model_fixed.py \
  "output/tandt_train_20260414_135259/point_cloud/iteration_30000/point_cloud.ply"
```

### 错误3：环境问题
```
ModuleNotFoundError: No module named 'plyfile'
```
**解决方案**：安装依赖
```bash
pip install plyfile numpy
```

## 高级用法

### 批量优化多个模型
```bash
#!/bin/bash
# batch_optimize.sh

MODELS=(
  "output/model1/point_cloud/iteration_30000/point_cloud.ply"
  "output/model2/point_cloud/iteration_30000/point_cloud.ply"
  "output/model3/point_cloud/iteration_30000/point_cloud.ply"
)

for model in "${MODELS[@]}"; do
  if [ -f "$model" ]; then
    echo "优化: $model"
    python optimize_model_fixed.py "$model" --method moderate
  else
    echo "文件不存在: $model"
  fi
done
```

### 创建优化前后对比
```bash
#!/bin/bash
# compare_optimization.sh

ORIGINAL="output/tandt_train_20260414_135259/point_cloud/iteration_30000/point_cloud.ply"
OPTIMIZED="output/tandt_train_20260414_135259/point_cloud/iteration_30000/point_cloud_optimized_fixed.ply"

# 分析原始模型
echo "=== 原始模型分析 ==="
python -c "
import numpy as np
from plyfile import PlyData

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

plydata = PlyData.read('$ORIGINAL')
vertices = plydata['vertex'].data
opacities = sigmoid(vertices['opacity'])
print(f'平均不透明度: {np.mean(opacities):.4f}')
print(f'不透明度<0.01: {np.mean(opacities<0.01):.2%}')
"

# 分析优化模型
echo "=== 优化模型分析 ==="
python -c "
import numpy as np
from plyfile import PlyData

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

plydata = PlyData.read('$OPTIMIZED')
vertices = plydata['vertex'].data
opacities = sigmoid(vertices['opacity'])
print(f'平均不透明度: {np.mean(opacities):.4f}')
print(f'不透明度<0.01: {np.mean(opacities<0.01):.2%}')
"
```

## 总结

### 推荐配置
- **一般情况**: 使用 `--method moderate --max-scale 0.5`
- **严重阴影**: 使用 `--method aggressive --max-scale 0.3 --remove-outliers`
- **轻微调整**: 使用 `--method conservative --max-scale 0.8`

### 输出文件
优化后的文件保存在同一目录，后缀为 `_optimized_fixed.ply`（默认）。
